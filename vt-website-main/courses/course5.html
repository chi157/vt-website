🎯 本課目標

學習如何將程式碼模組化、重複使用

掌握函數定義、參數傳遞與模組使用

📦 什麼是函數？

函數就像一個「程式碼盒子」，可以重複使用。

函數的優點
• 避免重複撰寫相同的程式碼
• 讓程式更容易閱讀和維護
• 方便測試和除錯
• 提高程式碼重用性

🔧 定義函數

使用 def 關鍵字定義函數。

1️ 基本函數
def greet(): print("你好！歡迎學習 Python") # 呼叫函數 greet() # 輸出：你好！歡迎學習 Python
⚠️ 函數名稱後要加括號 ()
⚠️ 函數內的程式碼要縮排
2 帶參數的函數
def greet(name): print(f"你好，{name}！") greet("柒柒") # 輸出：你好，柒柒！ greet("小明") # 輸出：你好，小明！
💡 參數讓函數更靈活
3 多個參數
def add(a, b): result = a + b print(f"{a} + {b} = {result}") add(5, 3) # 輸出：5 + 3 = 8 add(10, 20) # 輸出：10 + 20 = 30

↩️ 返回值

使用 return 讓函數回傳結果。

1️ 基本返回值
def add(a, b): return a + b result = add(5, 3) print(result) # 8
💡 return 會立即結束函數
2 多個返回值
def calculate(a, b): total = a + b diff = a - b return total, diff sum_result, diff_result = calculate(10, 3) print(sum_result) # 13 print(diff_result) # 7
3 條件返回
def check_age(age): if age >= 18: return "成年人" else: return "未成年人" status = check_age(20) print(status) # 成年人

⚙️ 參數類型

Python 支援多種參數類型。

1️ 預設參數
def greet(name, greeting="你好"): print(f"{greeting}，{name}！") greet("柒柒") # 你好，柒柒！ greet("柒柒", "嗨") # 嗨，柒柒！
💡 預設參數讓函數更靈活
2 關鍵字參數
def introduce(name, age, city): print(f"我是{name}，{age}歲，住在{city}") # 使用關鍵字參數，順序可以不同 introduce(age=25, name="柒柒", city="台北")
3 任意數量參數
def sum_all(*numbers): total = 0 for num in numbers: total += num return total print(sum_all(1, 2, 3)) # 6 print(sum_all(1, 2, 3, 4, 5)) # 15
💡 *args 可接收任意數量的參數

🌐 變數作用域

變數的「有效範圍」。

1️ 區域變數 vs 全域變數
x = 10 # 全域變數 def test(): y = 5 # 區域變數 print(x) # 可以讀取全域變數：10 print(y) # 5 test() # print(y) # 錯誤！區域變數在函數外無法使用
2 修改全域變數
count = 0 def increment(): global count # 宣告使用全域變數 count += 1 increment() print(count) # 1
⚠️ 盡量避免使用 global，改用返回值

📦 模組（Module）

模組是包含 Python 程式碼的檔案，可以重複使用。

1️ 匯入模組
import math print(math.pi) # 3.141592653589793 print(math.sqrt(16)) # 4.0
2 匯入特定功能
from math import pi, sqrt print(pi) # 3.141592653589793 print(sqrt(16)) # 4.0
3 使用別名
import math as m print(m.pi) # 3.141592653589793 print(m.sqrt(9)) # 3.0

🔢 常用內建模組：math

數學運算模組。

常用函數
import math print(math.ceil(4.3)) # 無條件進位：5 print(math.floor(4.7)) # 無條件捨去：4 print(math.pow(2, 3)) # 2 的 3 次方：8.0 print(math.sqrt(25)) # 平方根：5.0 print(math.factorial(5)) # 階乘：120

🎲 常用內建模組：random

隨機數產生模組。

常用函數
import random # 隨機整數（1-10） print(random.randint(1, 10)) # 隨機小數（0-1） print(random.random()) # 從列表隨機選擇 fruits = ["蘋果", "香蕉", "橘子"] print(random.choice(fruits))

📅 常用內建模組：datetime

日期時間處理模組。

常用功能
import datetime # 取得現在時間 now = datetime.datetime.now() print(now) # 格式化輸出 print(now.year) # 年 print(now.month) # 月 print(now.day) # 日 print(now.hour) # 時

📝 自訂模組

1️

建立自己的模組檔案。

建立模組檔案：mytools.py
def greet(name): return f"你好，{name}！" def add(a, b): return a + b
2 使用自訂模組
import mytools print(mytools.greet("柒柒")) print(mytools.add(5, 3))
💡 模組檔案要放在同一個資料夾

✍️ 實作練習

練習一：溫度轉換函數
寫一個函數 celsius_to_fahrenheit(c)
將攝氏溫度轉換為華氏溫度
公式：F = C × 9/5 + 32
練習二：計算平均分數
寫一個函數 calculate_average(*scores)
可接收任意數量的分數
回傳平均分數（四捨五入到小數點後1位）
練習三：密碼強度檢查
寫一個函數 check_password(pwd)
檢查密碼是否符合：長度 ≥ 8、包含數字、包含英文
回傳 True 或 False
練習四：骰子遊戲
使用 random 模組
寫一個函數擲骰子（1-6）
呼叫 3 次，計算總和
練習五：年齡計算器
使用 datetime 模組
輸入出生年份，計算並回傳年齡
提示：取得當前年份 - 出生年份
提示
記得使用 return 回傳結果
善用預設參數讓函數更靈活
模組要先匯入才能使用
